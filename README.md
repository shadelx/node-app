# Todo-App created with react, node.js and sqlite

A simple todo app made with node.js

## installation
For this project we use npm package manager
```
npm install

server
npm run dev

client
npm start
```

## dependencies
front
- React
- html css
back
- nodejs
- sqlite

## server properties

Routes:
- /api
- /api/todos (CRUD)
